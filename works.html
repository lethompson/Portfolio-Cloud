ABOUT THE PROJECT

The purpose of this project was to create a system that simplifies the way judges score student's science fair projects. Judges & coordinators were the main users of this application. The Reyes Elemetary Science Fair was a big event for the school, which made it a tedious process to be done by pen & paper. To solve this manual process, a full-stack web application was developed to score student's science fair projects & store the results in a MySQL database to be able to export a CSV file of the student's scores.
